# password-box
web app, flask, flask-sqlalchemy, sql, html, css, java


Password-box это веб-приложение предназначенное для хранение паролей от различных аккаунтов.

Цели разработки:
1) Научиться работать с библиотекой Flask
2) Научиться работать с библиотекой Flask-SqlAlchemy
3) Научиться работать с базами данных
4) Улучшить навыки front-end и back-end разработки
